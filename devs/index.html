List of 3DS homebrew developers

This is a full list containing info about all homebrew application developers.
I'll try to keep it up-to-date whenever possible, feel free to contribute if you want to.
Click on an entry to view details about that specific dev.

You can search devs by typing part of their name, location, or any other keyword.
For example: "Italy" will return all members living on Italy, whilst "cfw" will return all devs who work with CFW.
You can also sort by username, real name, and location.

Note: this is a WIP and some features are still missing.
